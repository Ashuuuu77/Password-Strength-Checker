 Password Strength Checker

A Python project that checks password strength and detects commonly used passwords to improve security awareness.

 Description
This project evaluates the strength of a password based on length, use of uppercase letters, numbers, and special characters. It also identifies commonly used passwords and warns the user.

  Features
- Password strength analysis (Weak / Moderate / Strong)
- Common password detection
- Simple and easy to use

 Tech Used
- Python

How to Run
Run the code and enter a password when prompted.
